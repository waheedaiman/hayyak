# Hayyak 
### AI Relocation Companion for Dubai Newcomers

Hayyak helps people moving to Dubai choose where to live, avoid relocation mistakes, and complete every setup step from housing to utilities.

## Features
- Onboarding quiz (budget, lifestyle, commute, household type)
- Neighbourhood Matchmaker — top 3 area recommendations
- Personalised A–Z Move-in Checklist
- Utility Setup Guide (DEWA, Ejari, e&, du, internet)
- AI Chatbot powered by Groq API

## Tech Stack
- LLM: Groq API (Llama 3, free tier)
- Frontend: Gradio
- Hosting: Hugging Face Spaces
